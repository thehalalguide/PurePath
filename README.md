\# 🌿 PurePath | The Halal Guide



\*\*PurePath\*\* is an open-source web platform designed to empower Muslims worldwide to make informed, faith-aligned decisions in a modern world. From checking food additives to understanding ethical finance, PurePath simplifies complex information into actionable clarity.



\## 🚀 Vision

To provide every Muslim with a digital companion that bridges the gap between traditional values and modern lifestyle choices.



\## ✨ Key Features

\* \*\*Halal Ingredient Checker:\*\* Searchable database for E-numbers and additives.

\* \*\*Ethical Finance:\*\* Simplified guides on modern banking, loans, and investments.

\* \*\*Multilingual Support:\*\* Fully responsive LTR (English) and RTL (Arabic) layouts.

\* \*\*Modern UI:\*\* Built with Tailwind CSS for a fast, mobile-first experience.



\## 🛠️ Built With

\* \*\*HTML5 / JavaScript\*\*

\* \*\*Tailwind CSS\*\* (Compiled for production)

\* \*\*Netlify\*\* (Continuous Deployment)



\## 🌍 For the Global Ummah

Whether you are in Rwanda, the UK, Indonesia, or anywhere else, PurePath is built to be your local guide to global standards.
